# Cuesta-Cougors
🍔 CS 233 — Online Food Delivery System
Welcome to our Java-powered food delivery system, where hungry customers, overworked drivers, and a surprisingly opinionated algorithm come together in beautiful OOP harmony.
Customers can order burgers, fries, and drinks. Drivers get assigned orders based on who has the highest rating (so yes, being nice does pay off). Orders are processed First-In, First-Out — just like a real drive-through, except this one actually works. Driver ratings are capped at the 10 most recent, because apparently even code has a short memory.
Under the hood: Queues keep orders fair, a Priority Queue keeps the best drivers at the front of the line, and a circular array makes sure old ratings get the boot. All user data persists between runs via file I/O, so nothing disappears — unlike your homework motivation at 11pm.
Built with Java, powered by caffeine, and submitted on time (probably).
